# 🍔 Food Store - Frontend Application
> **Evaluación 1 - Programación III - Tecnicatura Universitaria en Programación (UTN)**

Este proyecto consiste en una plataforma de e-commerce dinámica desarrollada íntegramente con **Vanilla TypeScript** y **Vite**. Se enfoca en la gestión de un catálogo de productos, interactividad mediante manipulación del DOM y persistencia de datos en el cliente.

---

## 🚀 Características Principales

* **Catálogo Dinámico:** Renderizado de productos desde estructuras de datos tipadas.
* **Sistema de Búsqueda:** Filtro por nombre con actualización en tiempo real.
* **Navegación por Categorías:** Clasificación de productos mediante etiquetas dinámicas.
* **Gestión de Carrito:** * Persistencia robusta mediante `localStorage`.
    * Control de stock (Validación de disponibilidad).
    * Lógica de acumulación (evita duplicados de ítems).
* **Formateo Localizado:** Precios presentados en formato moneda ARS (`es-AR`).

---

## 🛠️ Stack Tecnológico

* **Lenguaje:** [TypeScript](https://www.typescriptlang.org/)
* **Bundler:** [Vite](https://vitejs.dev/)
* **Gestor de paquetes:** `pnpm`
* **Estilos:** CSS3 (Diseño Responsivo)
